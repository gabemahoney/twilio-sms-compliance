## About Apiary SMS

Apiary SMS provides automated text message notifications for agent task completion, system alerts, and important updates.

## Opt-In

Users provide explicit consent by configuring their phone number in the Apiary application settings. By opting in, users acknowledge they will receive automated SMS notifications and can opt-out at any time by replying STOP to any message.

## Support

For questions or support, email support@apiary.example.com or reply HELP to any message.

---

Text **STOP** to cancel. Text **HELP** for help.

Message frequency varies. Message and data rates may apply.
